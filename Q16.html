3D on the Web, made efficient

Needle Engine

Creating spatial experiences for the web was never so fast and powerful. Bring your 3D content to everyone, on any device, with Needle Engine and our integrations for Unity and Blender.

Start with Needle today. Choose your workflow:

Built for Speed

Fast Iteration

Needle Engine builds on modern web technologies and toolchains. This makes iteration fast – you see changes in your browser in seconds.

Focus on your content instead of fighting with technology.

The Best of Both Worlds

Combine 2D and 3D

With Needle, you can combine 3D scenes with responsive and accessible HTML. You can use any front-end or full-stack framework, or just plain HTML and CSS.

For a lot of frontend and fullstack frameworks, we provide samples to get you started:

Everywhere Actions and Core Components

Packed with Features

Modern applications require a lot of building blocks. Needle provides dozens of core components out of the box that cover common interactions and functionality.

Our Everywhere Actions go even further – they work literally everywhere, including AR on iOS and visionOS, where no custom code can be used.

Learn more: Everywhere Actions
Learn more: Core Components

Lightmapping for the Web

Lightmaps

Learn more: Lightmaps

Materials and Shaders

Advanced Rendering

Needle supports the full PBR (physically-based rendering) workflow, including sophisticated effects like transmission, sheen, anisotropy and more.

Learn more: Physically-Based Rendering

Use Artist-Friendly Tools

Shader Graph Export

Custom shaders allow you to create your own style, and can be exported directly from Unity's Shader Graph. Work with textures, subgraphs, noise, vertex offsets & custom code.

Learn more: Custom Shaders

Automatic Optimization and Compression

Small Files, Light Runtime

We efficiently compress models, textures and animations for optimal performance, even on low-end devices. At runtime, your content loads in parts, to keep wait times short and pages snappy.

Learn more: Optimization and Compression
